  1. Server URL - http://13.127.139.107:8080

  3. The JSON response is by default coming in a property of "response".
  4. For example if the AJAX success method returns you Object 'data'
  5. You will have an encoded JSON in data.response
  6. Once you decode it using JSON.parse(data.response) you will get the actual properties
  7. For example the SUCCESS - RESPONSE JSON of Login is "response":{"success":true, "redirectTo":"/redirectUrl"}
  8. And you ajax success method is
  9. function ajaxsuccess(data) { //here data is the output returned from AJAX call}
  10. So to get the actual response
  11. Inside ajaxsuccess(data){var restResponse = JSON.parse(data.response);}
  12. The above line would be common to AJAX calls through out the project
  13. Now inorder to get the specific output for the AJAX call for Login
  14. var success = restResponse.success
  15. var redirectTo = restResponse.redirectTo
