Below are some instructions about JSON Response

  1. The JSON response is by default coming in a property of "response".
  2. For example if the AJAX success method returns you Object 'data'
  3. You will have an encoded JSON in data.response
  4. Once you decode it using JSON.parse(data.response) you will get the actual properties
  5. For example the SUCCESS - RESPONSE JSON of Login is "response":{"success":true, "redirectTo":"/redirectUrl"}
  6. And you ajax success method is
  7. function ajaxsuccess(data) { //here data is the output returned from AJAX call}
  8. So to get the actual response
  9. Inside ajaxsuccess(data){var restResponse = JSON.parse(data.response);}
  10. The above line would be common to AJAX calls through out the project
  11. Now inorder to get the specific output for the AJAX call for Login
  12. var success = restResponse.success
  13. var redirectTo = restResponse.redirectTo
